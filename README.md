## General Info

The Activity Finder is a full-stack app which takes a userâ€™s location and randomly suggests an activity for the user to do based on current real-time weather information. ## Setup

This program comes equipped with all necessary gems. To use them locally, execute the command:

`$ bundle`

To install and use the database, run the commands:

`$ rake db:seed `

and 

`$ rake db:migrate`

Run the program by: 

`rails s to start rails server`
`lite-server to serve the web app and open it in a browser`
